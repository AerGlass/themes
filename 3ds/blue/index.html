3DS Blue

I made a custom 3ds theme using photoshop cs6 and the 3ds theme editor.

you need a cfw 3ds (see https://3ds.hacks.guide)

turn off the console

remove the sd card

open the sd card on the pc

Download the zip file

Put the zip file on the themes folder of your sd card

remove the sd card from your pc

put the sd card on the 3ds

turn on the console

go to anemone (a 3ds theme mananger)

install the blue theme as usual

turn off the console

turn it back on

now it is applied
